To build and run Cell\+Viz, you will need\+:


\begin{DoxyItemize}
\item Boost dev libraries (not runtime)
\item Json\+Cpp dev libraries
\item Nlohmann Json dev libraries
\item CMake
\item SFML
\item Alpha Vantage API Key
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysubsubsection{\texorpdfstring{Installation on Ubuntu}{Installation on Ubuntu}}\label{md__r_e_a_d_m_e_autotoc_md2}

\begin{DoxyCode}{0}
\DoxyCodeLine{sudo\ apt\ update}
\DoxyCodeLine{sudo\ apt-\/get\ install\ libboost-\/all-\/dev\ libjsoncpp-\/dev\ nlohmann-\/json3-\/dev\ libsfml-\/dev}

\end{DoxyCode}


To use the Alpha Vantage API key, set the environment variable\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{export\ API\_KEY=YOUR\_ALPHA\_VANTAGE\_KEY}

\end{DoxyCode}
